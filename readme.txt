== Description ==

Easily embed popular hotel booking engines on any Wordpress site

= Plugin Features =

*   Block to embed hotel booking engines (e.g. CloudBeds, MyAllocator, Siteminder, Wubook).
*   Simply add the Block and paste your Booking Engine URL/Link for your booking engine in the Block Settings.
*   Use the default configuration, or customize for your site requirements.

== Installation ==

1.  Install plugin from the wordpress directory or upload using admin screens
2.  Activate the plugin through the 'Plugins' menu in the WordPress Admin area.
3.  Edit an existing Page or Add New and use "+" sign to search for your booking engine provider
